# Medicare Plus Healthcare Portal

A healthcare management web application developed for managing appointments, doctors, patients, and healthcare services.

## Features
- Doctor appointment booking
- Admin dashboard
- Doctor & patient management
- Feedback/contact system
- Responsive web interface

## Technologies
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

## Run Locally
1. Install XAMPP
2. Move project into `htdocs`
3. Start Apache & MySQL
4. Import database
5. Open:
```txt
http://localhost/medicare_plus/
```
