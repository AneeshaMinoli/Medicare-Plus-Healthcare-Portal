# Medicare Plus Healthcare Portal

A modern healthcare management web application developed for managing doctor appointments, patient services, healthcare operations, and medical information through a clean and user-friendly interface.

---

## Features

- Patient appointment booking system
- Doctor management dashboard
- Admin dashboard
- Patient management
- Appointment tracking
- Medical information pages
- Contact and feedback system
- Responsive healthcare UI
- File upload functionality
- Authentication and management features

---

## Technologies Used

- PHP
- HTML5
- CSS3
- JavaScript
- MySQL
- XAMPP

---

## Modules

### Patient Features
- Book appointments
- View doctors
- Access healthcare services
- Send feedback and contact messages

### Admin Features
- Manage doctors
- Manage appointments
- Manage patient records
- Monitor healthcare activities

### Doctor Features
- Appointment handling
- Patient management
- Medical report management

---

## Project Structure

```txt
config/         -> Database configuration
uploads/        -> Uploaded files and images
*.php           -> Backend logic
*.html          -> Frontend pages
*.png / *.jpg   -> Images and UI assets
```

---

## How to Run

1. Install XAMPP
2. Move the project folder into:

```txt
htdocs/
```

3. Start:
- Apache
- MySQL

4. Import the database using phpMyAdmin

5. Open browser and run:

```txt
http://localhost/medicare_plus/
```
