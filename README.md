# Medicare Plus Healthcare Portal

A healthcare management web application for doctor appointments, patient services, and healthcare management operations.

## Features
- Appointment booking system
- Admin dashboard
- Doctor management
- Patient management
- Contact & feedback system
- Responsive UI

## Technologies Used
- PHP
- HTML
- CSS
- JavaScript
- MySQL
- XAMPP

## How to Run
1. Install XAMPP
2. Move project into `htdocs/`
3. Start Apache & MySQL
4. Import database
5. Open:
```txt
http://localhost/medicare_plus/
```
